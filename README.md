# TechWave Conference 2025

A modern event landing page for **TechWave 2025**, built with HTML and CSS.  
It provides a professional interface for showcasing event details, schedules, speakers, and includes a functional contact form for attendee registration. This project highlights clean design, responsive layout, and front‑end development skills.

---

## Features
- Responsive layout for desktop and mobile  
- About section introducing the conference  
- Organized event schedule with sessions and timings  
- Speaker profiles highlighting key experts  
- Contact form for attendee registration  
- Footer credit line: Made by Prince

## Installation
1. Go to the repository: [TechWave Event Page](https://github.com/Prince-of-Ideas/Tech-event-registration-page)  
2. Click on **Code** and copy the repository link.  
3. Open your terminal and run:  
   ```bash
   git clone https://github.com/Prince-of-Ideas/Tech-event-registration-page.git
   ```
4. Open the project folder and double‑click `index.html` to view the page in your browser.

## Technologies Used
